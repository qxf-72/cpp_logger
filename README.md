# C++ Linux 多线程日志系统

## 项目简介

本项目是一个基于 C++11 实现的 Linux 多线程日志系统，支持日志级别过滤、日志时间戳、线程 ID、源码文件名与行号输出，并使用互斥锁保证多线程环境下日志写入的线程安全。

该项目用于练习 C++ 工程开发中的类封装、单例模式、RAII、文件 IO、多线程同步、CMake 构建等内容。后续计划在当前同步日志器基础上扩展异步日志、阻塞队列、日志文件滚动和性能测试功能。

## 当前功能

- 支持日志写入文件
- 支持 DEBUG / INFO / WARN / ERROR 四种日志级别
- 支持日志级别过滤
- 支持输出当前时间
- 支持输出线程 ID
- 支持输出源码文件名和行号
- 支持宏调用方式，例如 `LOG_INFO("message")`
- 使用 `std::mutex` 保证多线程写日志安全
- 使用 CMake 构建项目
